Budokan Thiais

J'ai eu l'occasion de travailler sur ce site lors de mon stage aupres de l'association d'arts martiaux Budokan.
Mon but etais d'ajouter des fonctionnalitees a la page admin du site. Ce site a ete cree sur le framework Laravel par des eleves de secondes annee qui m'ont precedes.
Ce stage m'a donc permis d'apprendre le fonctionnement des frameworks, ainsi que d'analyser du code plus avance et plus consequent. J'ai donc pu ajouter differents onglets suplementaires a la page admin.
Ces onglets sont : Page d'appel, Inscription aux disciplines et la page des cours.
Sur ces onglets je peut voir les differents elements stockes dans la base de donnee, en rajouter, supprimer et modifier.

Elements du site :


Page admin

Page admin crée par les éleves de 2nde année

Appel

Page admin permettant de créer une fiche d'appel grace a un formulaire

Cours

Page admin permettant de gérer les cours du club

Inscriptions

Page admin permettant de gérer les inscriptions aux disciplines
