Bonbons

Ce site a ete realise pour apprendre la communication entre le PHP et une Base de donnees. Permettant ainsi de stocker des bonbons, les afficher et de les gerer.
Une page admin devait etre creer afin de gerer les bonbons de la base de donnee et ainsi pouvoir en ajouter, suprimer et modifier.
Un panier devais etre ajoute a la fin du travail, et le site fut aussi adapte a la norme de projet MVC avec les controlleurs modeles et vues.

Elements du site :


Page bonbons

La page des bonbons affiche tout les bonbons dans la base de donnees

Modifications

Via le boutton "modification BDD" sur la page bonbon, il est possible de faire la gestion des bonbons
