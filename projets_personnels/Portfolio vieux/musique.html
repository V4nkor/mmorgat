Musique

Ce site a ete realise lors d'une evaluation qui durais 4h. Cette evaluation avais pour but de controller nos connaissances apprises lors des differents cours de php.
Ma consigne lors de cette evaluation etais d'implementer certains elements au site tel que l'affichage des musiques de la base de donnees, de pouvoir les noter, voir le classement, avoir une page d'informations sur le titre, ainsi qu'une recherche par artiste.

Elements du site :


Page d'acceuil

La page d'acceuil affiche toutes les musiques de la base de donnees

Noter

Via le boutton "noter" sur l'acceuil, il est possible de donner une note a une musique via un formulaire

Classement

Les notes sont comptabilisees et un classement des albums est disponible

Informations

La page d'information d'un album est accessible via le classement, en cliquant sur le titre
