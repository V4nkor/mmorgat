Valve Fansite :

Ce site est un exercice d’apprentissage de Bootstrap. J’ai choisi comme sujet la société de jeux vidéos Valve.
Pour cet exercice j’ai utilisé de nombreuses ressources Bootstrap comme les cards, les buttons, les jumbotrons, les carousels, ainsi que les forms.
