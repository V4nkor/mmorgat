Gallimedia :

J'ai effectué ce stage dans l'entreprise multimédia Gallimedia qui est un prestataire de sites web pour les communes.
Ce stage, d'une durée de sept semaines, s'est déroulé dans un premier temps en distanciel, puis en semi-présentiel.
Pour le travail en distanciel, j'ai utilisé le logiciel de communication slack.
J'ai eu l'occasion de travailler sur différents sujets, ce qui m'a permis de beaucoup apprendre, aussi bien dans le domaine informatique que sur la vie de l'entreprise.

Déroulement :

Au début de mon stage après avoir été invité sur slack, j'ai mis en place une connection SSH vers un serveur distant personnel créé à l'occasion de ce stage via le logiciel PuTTY, après que mon tuteur de stage ait whitelisté mon adresse IP publique sur la machine.
Ayant travaillé durant tout le stage via SSH vers une machine virtuelle Linux distante, j'ai acquis de meilleures connaissances sur le fonctionnement de Linux et ses commandes.
J'ai également beaucoup appris sur les packets et le paramétrage de Linux, car ma première mission fut de rendre opérationnels les packets installés sur la machine. Ensuite, j'ai eu pour mission de réaliser des scripts bash d'automatisation, afin de créer une machine Linux "from scratch".
Le script s'occupe d'installer les packets essentiels, puis de les paramétrer.
